RoofMapper


A tool that makes contributing to OpenStreetMap really accessible.

 

Running at www.roofmapper.eu.

 

This is an experiment in offering a way for non-computer-savvy people to contribute to the project. Here the user just has to select the building roof material, with the help of the aerial imagery, by clicking the corresponding button. When finished, the user clicks “upload” to send changes, and that’s all. Everything else, from the selection of the buildings to the changeset message, is taken care of automatically.

The tool is designed to be used with tablets, though it is fully usable on a desktop computer. (It can be used using the keyboard only.)

RoofMapper can also be used by seasoned mappers for quick contributions.
