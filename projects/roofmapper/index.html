RoofMapper


A tool that makes contributing to OpenStreetMap really accessible.

 

This is an experiment in offering a way for non-computer-savvy people to contribute to the project. Here the user just has to select the building roof material, with the help of the aerial imagery, by clicking the corresponding button, and click upload to send changes. Everything else, from the selection of the buildings to the changeset message, is taken care of automatically.

The tool is designed to be used with tablets, though it is fully usable on a desktop computer. (It can be used using the keyboard only.)

RoofMapper can also be used by seasoned mappers for quick contributions.

 

No public server is currently running! (The tool is ready, I just need to sort out hosting issues. (^^;) )
